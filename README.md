# pokedExpress
Projet réalisé au cours de ma formation chez O'Clock.

## Objectifs
Concevoir une application web qui permet de :
- Lister tous les Pokémon de la base.
- Afficher les détails d'un Pokémon, y compris son type et ses statistiques.
- Filtrer les Pokémon par type.

## Instructions
Voici la liste des pages à réaliser, avec un aperçu de chacune dans le dossier **resultat** :
- **Page d'accueil** : Liste tous les Pokémon de la base (home.png).
- **Page de détail d'un Pokémon** : Affiche son type et ses statistiques (detail.png).

## Outils utilisés
- **Node.js**
- **HTML et CSS**
- **npm**
- **Express**

Un fichier JSON, `pokedex.json`, est fourni dans le dossier **data**. Ce fichier contient la liste des Pokémon ainsi que leurs types.
